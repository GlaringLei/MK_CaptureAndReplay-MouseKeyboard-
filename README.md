# MK_CaptureAndReplay-MouseKeyboard-
基于QT5.14.2_MSVC2017_32bit的环境，采用多线程编程开发了一款能够捕获用户在当前电脑环境下的鼠标键盘事件，并采用JSON格式记录，同时也可以选择指定目录下加载先前用户记录的事件记录数据，配置事件回放参数，可以完成对先前操作的复现实现无聊琐碎的工作（ps: Ciallo～ (∠・ω&lt; )⌒★别太指望能够完全复现，很有可能由于当前电脑环境不同于之前的而导致回放失败，比如之前想要打开b站，但是由于现有网速较慢，导致网页没能打开，后续的操作没能衔接上而失败，问题不大但是点错到其他地方就不好了，慎用！）。
